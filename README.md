# ЛВТ Производство — Telegram Бот + Веб-панель

Закрытая система учёта производства ЛВТ-ламината с авторизацией по кодам.

---

## Как работает закрытый доступ

**Три роли, три кода:**

| Роль | Код по умолчанию | Что может |
|------|-----------------|-----------|
| Миксерщик | `mix2026` | Вносить замесы |
| Оператор экструзии | `ext2026` | Вносить паллеты, замечания |
| Руководитель | `boss2026` | Всё: отчёты, задания, рецептура, управление пользователями |

**Логика входа:**
1. Сотрудник открывает бота → `/start`
2. Бот просит код доступа
3. Сотрудник вводит код → Telegram ID сохраняется
4. Больше код вводить не нужно — бот узнаёт по ID

Без кода бот ничего не покажет. Чужой человек увидит только запрос кода.

---

## ВАЖНО: поменяй коды после первого запуска

Дефолтные коды опубликованы в README — их знает любой.

1. Сам войди с кодом `boss2026` (получи роль руководителя)
2. В меню → `🔑 Коды доступа`
3. Отправь команды:
   ```
   /code mixer МойНовыйКод123
   /code extruder ДругойКод456
   /code boss СекретныйКод789
   ```
4. Раздай новые коды сотрудникам

**Отозвать доступ сотруднику:**
- Меню → `👥 Пользователи` → скопируй его Telegram ID
- Отправь: `/revoke 123456789`

## ШАГ 1 — Создать бота

1. В Telegram найди **@BotFather**
2. `/newbot` → имя → username
3. Сохрани токен вида `1234567890:ABCdef...`

---

## ШАГ 2 — Деплой на Railway (бесплатно)

### 2.1 Регистрация
Зайди на **railway.app** → войди через GitHub.

### 2.2 Загрузка кода
1. **github.com** → New repository → `lvt-bot` → Create
2. Перетащи все файлы из архива прямо в браузер

### 2.3 Проект в Railway
1. **New Project** → **Deploy from GitHub repo** → `lvt-bot`
2. Railway обнаружит Procfile и создаст сервисы `web` и `bot`

### 2.4 Переменные окружения (Variables)
```
BOT_TOKEN=твой_токен_от_BotFather
SECRET_KEY=любая_случайная_строка_32_символа
```

### 2.5 Домен веб-панели
Settings → Domains → **Generate Domain**
Копируй URL и добавь в Variables:
```
WEB_URL=https://lvt-bot.up.railway.app
```

---

## ШАГ 3 — Первый вход

1. Открой бота → `/start`
2. Введи `boss2026` → ты руководитель
3. **Сразу поменяй все 3 кода**
4. Раздай новые коды сотрудникам

---

## Команды руководителя

| Команда | Что делает |
|---------|-----------|
| `/recipe Компонент кг` | Изменить рецептуру (`/recipe Дробленка 120`) |
| `/code роль новый_код` | Сменить код (`/code mixer новый123`) |
| `/revoke TG_ID` | Отозвать доступ сотрудника |
| `/done` | Завершить ввод недельного задания |
| `/start` | Главное меню |

## Локальный запуск

```bash
cd lvt_bot
pip install -r requirements.txt
cp .env.example .env
# заполни BOT_TOKEN в .env
python run.py
```

Веб: http://localhost:5000
Бот: работает в Telegram

---

## Веб-панель — разделы

- **Отчёт** — выпуск, брак, замесы, сырьё за дату
- **Неделя** — прогресс выполнения недельного задания
- **Миксер** — история замесов
- **Экструзия** — история паллет
- **Задание** — создание недельного плана
- **Рецептура** — редактор состава

**Совет:** можно добавить логин/пароль на веб-панель. Скажи — допишу.
